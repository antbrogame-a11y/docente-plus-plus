Breadcrumbs & Floating AI Assistant

navigation Breadcrumbs Component

A mobile-first, responsive breadcrumb navigation component with accessibility features. It automatically builds the breadcrumb trail from URL parameters and query strings.

Features:
  • Mobile-first responsive design
  • Accessibility-first with aria-current support
  • Auto-hides on Home page
  • Router/query parameter-based generation
  • Dark mode and high contrast support
  • Keyboard navigation
// Usage Example:
import { initBreadcrumbs } from './src/components/breadcrumbs/breadcrumbs.js';

// Initialize
initBreadcrumbs('breadcrumbs-container');

// Or manually update
const breadcrumbs = getBreadcrumbs();
breadcrumbs.update([
  { label: 'Home', href: 'index.html', isCurrent: false },
  { label: 'In Classe', href: null, isCurrent: true }
]);

psychology Floating AI Assistant

A responsive AI assistant panel that appears as a full-screen modal on mobile and a right drawer on desktop. Includes voice input (mock), text chat, and accessibility features like focus trap and screen reader support.

Features:
  • Mobile: Full-screen modal
  • Desktop: Right-side drawer
  • Text input with auto-resize
  • Mock voice recording with transcription
  • Mock AI responses with typing indicator
  • ESC key to close
  • Focus trap for accessibility
  • aria-live regions for screen readers
  • TODO comments for real API integration
// Usage Example:
import { initFloatingAssistant } from './src/ui/floating-assistant/floating-assistant.js';

// Initialize
initFloatingAssistant();
